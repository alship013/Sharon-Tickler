# Contributing to Sharon Tickler

We welcome contributions from everyone! Here's how you can contribute:

## How to Contribute

### For Users
- **Fork the repository**: Click the "Fork" button on GitHub
- **Make changes**: Edit files in your forked repository
- **Submit changes**: Create a Pull Request to merge your changes

### Direct Push Access
This repository allows direct pushes for:
- Repository collaborators
- Organization members with write access

## Getting Started
1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/Sharon-Tickler.git`
3. Create a branch: `git checkout -b feature-name`
4. Make your changes
5. Push to your fork: `git push origin feature-name`
6. Create a Pull Request

## What to Contribute
- Add new random facts to the `randomFacts` array in `script.js`
- Improve UI/UX in `style.css` and `index.html`
- Fix bugs or enhance functionality
- Update documentation

## Guidelines
- Follow the code style in `AGENTS.md`
- Test your changes before submitting
- Keep pull requests focused and concise

## Questions?
Feel free to open an issue for questions or suggestions!
